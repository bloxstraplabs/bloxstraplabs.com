To uninstall Bloxstrap, you uninstall it like any other program on your computer. Here's how:

<Steps>

1. Open the Settings app
   

   ![Screenshot showing 'Settings' being searched for in Windows Search](../../../../assets/wiki/uninstall-bloxstrap-1.png)

2. Go to the Apps section. (On Windows 11, look in the sidebar for this)

   ![Screenshot showing the 'Apps' menu item in the Settings main screen](../../../../assets/wiki/uninstall-bloxstrap-2.png)

3. Search for Bloxstrap, select it and click Uninstall
   
   ![Screenshot showing Bloxstrap being searched for in the Windows Settings apps list with an uninstall button visible](../../../../assets/wiki/uninstall-bloxstrap-3.png)

</Steps>
